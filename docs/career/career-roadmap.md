## Skills Employers Look For

### Technical Skills (Must Have)

- **Networking:** TCP/IP, OSI model, packet analysis
- **Linux:** Comfortable in command line
- **Scripting:** Python and/or Bash
- **Security concepts:** CIA triad, OWASP, encryption, auth
- **Tools:** Nmap, Wireshark, Burp Suite (at minimum)

### Proof of Skills (Sets You Apart)

- 🏆 CTF competitions — with solved challenges to show
- 🌐 Bug bounty — even low-severity findings count
- 🖥️ Home lab — VMs, vulnerable machines
- 📝 Blog / writeups — document your learning
- 🎓 Certifications — especially practical ones
- 💻 GitHub — scripts, tools, CTF solutions

## 🏅 Certifications Roadmap

### Free / No-Cost Certifications

```
✅ Google Cybersecurity Professional Certificate (Coursera — audit free)
✅ IBM Cybersecurity Analyst Professional Certificate (Coursera — audit free)
✅ TryHackMe Certificates (free after completing paths)
✅ Cisco Networking Academy — Cybersecurity courses
✅ SANS Cyber Aces (free)
```

### Paid Certifications (Priority Order for Students)

```
LEVEL 1 — START HERE:
  ├── CompTIA Security+ (~$380, widely recognized globally)
  └── eJPT — eLearnSecurity Jr. Penetration Tester (~$200, practical)

LEVEL 2 — INTERMEDIATE:
  ├── PNPT — Practical Network Penetration Tester (~$400, TCM Security)
  ├── CompTIA PenTest+
  └── CEH — Certified Ethical Hacker (~$1,100 exam only)

LEVEL 3 — ADVANCED:
  ├── OSCP — Offensive Security Certified Professional (~$1,499, gold standard)
  ├── CRTO — Certified Red Team Operator (~$400)
  └── OSED, OSEP (advanced Offensive Security)

LEVEL 4 — SPECIALIZED:
  ├── GPEN, GWAPT — GIAC certs (expensive, ~$2,500+)
  ├── AWS Security Specialty
  └── CISSP — management/leadership ($700)
```

> 💡 **Tip for students:** Start with free resources and TryHackMe paths. The eJPT is the most affordable practical cert for beginners (~$200). OSCP is the ultimate goal for pentesters.

## Building Your Portfolio

### Step 1: Create a GitHub Profile

```markdown
Showcase:
- CTF writeups (post-competition)
- Python/Bash scripts you wrote
- Security tools you built
- Notes and cheat sheets
- Your learning journey
```

### Step 2: Write Writeups

After every CTF or lab you complete:
1. Write a detailed writeup
2. Post it on GitHub or a blog (Medium, Hashnode)
3. Share on LinkedIn

### Step 3: Build a Home Lab

```
Recommended VMs:
- Kali Linux (attacker)
- Metasploitable2 (vulnerable target)
- DVWA (vulnerable web app)
- Windows Server (for AD practice)
- TryHackMe / HTB (cloud labs, no hardware needed)
```

### Step 4: Participate in Bug Bounties

1. Start at [HackerOne](https://hackerone.com) — look for programs with broad scope
2. Focus on one vulnerability type to start (IDOR is beginner-friendly)
3. Read disclosed reports to learn
4. Even P5 (low severity) findings matter for your profile

## Timeline Suggestion for IIST Students

```
Year 1 (Freshman):
  → Complete Phases 1–4 of this roadmap
  → Finish TryHackMe Pre-Security path
  → Complete OverTheWire: Bandit
  → Participate in beginner CTFs (PicoCTF)

Year 2 (Sophomore):
  → Complete Phases 5–8
  → Finish TryHackMe Jr Pentester path
  → Complete PortSwigger Web Academy (Apprentice level)
  → Participate in college CTF (IIST CTF!)
  → Start bug bounty hunting

Year 3 (Junior):
  → Complete Phase 9 (Advanced Topics)
  → Earn eJPT or PNPT certification
  → Participate in national/international CTFs
  → Build public CTF writeup portfolio
  → Internships in cybersecurity

Year 4 (Senior):
  → Target OSCP if interested in red team
  → Full-time bug bounty or internship-to-job
  → Build network (LinkedIn, security conferences)
  → Campus placement or off-campus jobs
```
