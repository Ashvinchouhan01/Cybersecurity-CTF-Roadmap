## вҡҷпёҸ Reverse Engineering Basics

> рҹ“Ғ Full notes вҶ’

### What is Reverse Engineering?

Reverse engineering in CTF means analyzing a binary/program **without source code** to understand what it does and find the flag.

### Quick Start Checklist

- [ ] Learn x86/x64 assembly basics
- [ ] Understand calling conventions (cdecl, fastcall)
- [ ] Learn how to use `strings`, `file`, `xxd` commands
- [ ] Practice with Ghidra (decompiler)
- [ ] Learn GDB for dynamic analysis
- [ ] Understand stack layout and function calls
- [ ] Practice on PicoCTF reverse engineering challenges

### Key Commands

```bash
file binary          # What type of file?
strings binary       # Human-readable strings
xxd binary | head    # Hex dump
ltrace ./binary      # Library calls
strace ./binary      # System calls
objdump -d binary    # Disassemble
gdb binary           # Debug
```

## рҹ”җ Cryptography Basics

> рҹ“Ғ Full notes вҶ’ 

### Common CTF Ciphers

| Cipher | How to Identify | Tool to Break |
|---|---|---|
| Caesar | Shifted alphabet | CyberChef ROT brute |
| ROT13 | Looks like gibberish text | CyberChef |
| Base64 | Ends with `==`, uses A-Z/a-z/0-9 | `base64 -d` or CyberChef |
| XOR | Repeating pattern, key reuse | XORtool, Python |
| Vigenere | Polyalphabetic | dcode.fr |
| RSA | `n`, `e`, `c` given | RsaCtfTool |
| MD5/SHA | 32/40/64 hex chars | hashcat, crackstation.net |

### Python Crypto Snippets

```python
# Base64 decode
import base64
base64.b64decode("SGVsbG8=")

# XOR two strings
xor = lambda a, b: bytes(x^y for x,y in zip(a,b))

# MD5 hash
import hashlib
hashlib.md5(b"password").hexdigest()

# RSA with small e (cube root attack)
from sympy import integer_nthroot
m, _ = integer_nthroot(c, e)
print(bytes.fromhex(hex(m)[2:]))
```
